List of HTML Tags

  1. <html>

    The root element of an HTML document that defines the entire structure of the page.

  2. <head>

    Contains meta-information about the document, such as title, character encoding, and links to external resources.

  3. <title>

    Sets the title of the web page, which is displayed in the browser's title bar or tab.

  4. <meta>

    Provides metadata about the HTML document, like character encoding, author, description, and more.

  5. <link>

    Establishes a connection between the current document and external resources, such as stylesheets.
